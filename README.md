### Technologies Used
* Swift
* UIKit
* [Lottie for iOS](https://github.com/airbnb/lottie-ios)

This app is powered by data provided by [Zomato API](https://developers.zomato.com/).

### Usage
Make an `APIKey.swift` file and put your Zomato API key in a `value` there.
